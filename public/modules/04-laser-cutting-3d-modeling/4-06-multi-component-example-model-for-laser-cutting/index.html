04.06 Multi Component Example Model for Laser cutting

This video walks through a simple phone stand with interlocking pieces. It uses offset planes and mirroring. Each panel of the stand is made as a separate component so the design history stays clean and the pieces can be layed flat. This example uses a previous method of laying parts flat with joints. Now Fusion 360 uses the Modify > Arrange Tool to lay parts flat.
